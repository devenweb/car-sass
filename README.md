# Royal Car Rental (Drive Mauritius)
A high-fidelity, automated car rental platform built for enterprise-grade fleet management and seamless customer booking experiences.

## 🚀 Key Modules
- **Web App**: Premium marketplace with live invoicing and precision booking.
- **Admin App**: High-density Command Center for fleet, pricing, and operational management.
- **Supabase Backend**: Scalable infrastructure with real-time data sync and security.

## 🛠️ Technology Stack
- **Frontend**: Next.js 14, Tailwind CSS, Lucide React.
- **Backend**: Supabase (Auth, DB, Storage).
- **State**: React Context + Hooks.
